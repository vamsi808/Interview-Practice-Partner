# **App Name**: Ace the Interview

## Core Features:

- Role Selection: Users can select a specific job role (e.g., software engineer, sales manager) to tailor the mock interview.
- Mock Interview Conductor: The application conducts mock interviews, posing questions relevant to the selected role. Use speech recognition if available.
- Dynamic Follow-Up Questions: The AI dynamically generates follow-up questions based on the user's responses, simulating a natural conversation and probing deeper into their experience and qualifications. This involves the use of an LLM tool.
- Performance Assessment: Post-interview, the AI analyzes the user's responses, evaluating their communication skills, technical knowledge, and overall performance based on an internal rubric that it uses as a tool. Reports the interview outcomes back to the user.
- Personalized Feedback: Provides personalized feedback highlighting strengths and areas for improvement. This uses the LLM as a tool to incorporate relevant best practices in similar interview scenarios.

## Style Guidelines:

- Primary color: A professional and confident dark blue (#243A73) to inspire trust and readiness.
- Background color: A light, calming gray (#E9ECEF) to reduce distraction.
- Accent color: A motivational and encouraging green (#34A0A4) for positive reinforcement.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look, suitable for headlines or body text.
- Use clear, professional icons to represent different roles, feedback types, and functionalities.
- A clean, structured layout with clear visual hierarchy to facilitate easy navigation and focus on the interview questions and feedback.
- Subtle transitions and animations to indicate progress and provide feedback, enhancing user engagement without being distracting.
